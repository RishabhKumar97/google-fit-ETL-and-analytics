Google Fit Metrics API

Available endpoints:

1. GET /metrics

Response: List of all available metrics.

2. GET /metrics/<metric_name>

Query parameters:

    {
    "entity": "entity_name"   (required, comma-separated for multiple),
    "start_date": "YYYY-MM-DD" (optional),
    "end_date": "YYYY-MM-DD"   (optional)
    }

Response: Metric data in JSON format.
